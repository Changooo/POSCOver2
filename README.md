## Installation

### Prerequistes

- python3

### 1. Clone

```shell
$ git clone https://github.com/2020-SKKU-POSCO-International/POSCO.git
```

### 2. Install Packages

```bash
$ pip3 install requirements.txt
```

### 3-1. Start development server

```shell
$ python3 app.py
```

서버 시작 후,
[`http://localhost:5001`](http://localhost:5001) 에 접속
